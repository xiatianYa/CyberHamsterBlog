# 赛博仓鼠博客

#### Description
赛博仓鼠博客(CyberHamsterBlog)，一个基于SpringBoot架构的前后端分离博客系统。Web端使用Vue + Element , 后端使用 Spring boot + mybatis-plus + redis进行开发，使用 So-Token做登录验证和权限校验，文章存储采用阿里云Oss,使用ElasticSearch作为全文检索服务,
